# 🤝 人机协作任务规划器

## 此演示展示了什么

此演示展示了 CopilotKit 的人机交互功能：

1. 协作规划：Copilot 会生成任务步骤，并让您决定执行哪些步骤。
2. 交互式决策：选择或取消选择步骤以自定义执行计划
3. 自适应响应：副驾驶会根据您的选择调整其执行方式，甚至可以处理遗漏的步骤。

## 如何互动

请按照以下步骤体验演示：

1. 请您的副驾驶协助完成某项任务，例如：
   - 给我做个三明治
   - “计划一次周末旅行”
   - “组织一场生日派对”
   - “开始种菜”
2. 请查看副驾驶提供的建议步骤
3. 使用复选框选择或取消选择步骤以自定义计划
   - 尝试移除一些必要步骤，看看副驾驶如何适应！
4. 点击“执行计划”查看根据您的选择得出的结果。

## ✨ 人机协作的魔法在行动中

技术上发生了什么：

- 代理会分析您的请求并将其分解为逻辑步骤。
- 这些步骤将通过动态 UI 组件呈现给您。
- 您的选择将作为用户输入记录下来。
- 代理人在执行计划时会考虑您的选择。
- 该代理人通过创造性的问题解决能力来适应缺失的步骤。

本次演示内容：

- 副驾驶会为你的任务提供详细的分步计划。
- 您可以完全控制要包含哪些步骤。
- 如果省略必要步骤，Copilot 会提供有趣且富有创意的替代方案。
- 最终的执行结果反映了你的选择，展现了人为因素如何影响最终结果。
- 每条回复都根据您的具体选择量身定制。

这种人机协作模式创造了一种强大的协作体验，人类的判断和人工智能的能力共同作用，取得了比任何一方单独行动都能取得的更好结果！
